# KOTOSIO ウエディング LP - セッションセーブ

## プロジェクト概要
KOTOSIOウエディングのランディングページ（LP）デザイン・開発プロジェクト

## 理念
「土地が語り、写真が遺し、人生の"ことしお"になる結婚式を。」

## ブランドの中心軸
「ただいま、と言える場所で、家族が生まれる。」

## ターゲット
- **年齢**: 26〜45歳
- **世帯年収**: 700万〜1500万
- **対象地域**: 駒ヶ根・豊橋・湘南・横浜に「ただいま」と言える場所を持つカップル

## デザイン方向性
- **トーン**: Landscape感重視（海と山）、さわやかな配色
- **デバイス**: SPファースト

## 最新カラーパレット
- --primary: #2a4a52（水色ベースのダーク）
- --yokohama: #5BA3B8（横浜・湘南・海）
- --komagane: #8BAF7A（駒ヶ根・森）
- --cream: #f5f0eb

## カラーバランス目標
- 白・クリーム（60%）/ 横浜の水色（30%）/ 駒ヶ根の薄緑（10%）

## 技術スタック
- HTML5 / CSS3 / Vanilla JavaScript
- GitHub × Netlify 自動デプロイ
