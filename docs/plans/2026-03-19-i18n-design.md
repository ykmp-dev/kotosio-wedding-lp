# 多言語化（日英切り替え）設計書

## 概要

KOTOSIO Wedding LPに日本語/英語の切り替え機能を追加する。

## ターゲット

- 日本在住の外国人カップル
- 海外在住で日本に来て撮影したいカップル

## アプローチ

`data-i18n`属性 + JSON翻訳ファイル方式を採用。

## ファイル構成

```
KOTOSIOLP/
├── index.html          ← data-i18n属性を追加
├── lang/
│   ├── ja.json         ← 日本語テキスト（マスター）
│   └── en.json         ← 英語翻訳
└── js/
    └── i18n.js         ← 言語切り替えロジック
```

## JSONキー構造

- セクション単位でグループ化（2階層: セクション名.要素名）
- キー名は英語
- `data-i18n="hero.title_1"` のようにHTMLと1:1対応

### 翻訳対象の属性

| HTML属性 | 用途 |
|---------|------|
| `data-i18n="key"` | テキスト内容 |
| `data-i18n-placeholder="key"` | input/textareaのplaceholder |
| `data-i18n-alt="key"` | imgのalt |

## 言語切り替えUI

- ヘッダーナビ右端に `[JP | EN]` トグルボタン
- スマホ時もヘッダー右上に常時表示（ハンバーガー外）
- アクセント色 #8b7355 で選択状態を表示

## 言語判定ロジック

1. `localStorage('lang')` を確認 → あればそれを使用
2. なければ `navigator.language` で判定（ja系→日本語、それ以外→英語）
3. 切り替え後は `localStorage` に保存
4. 切り替え演出はなし（瞬時）

## フォールバック

JSON読み込み失敗時はHTML内の日本語テキストがそのまま表示される。

## 更新ワークフロー

1. 日本語HTMLを更新
2. `lang/ja.json` を更新
3. Claude Codeに `lang/en.json` の翻訳更新を依頼

## コンテンツ範囲

全ページ・全セクションを完全翻訳。
